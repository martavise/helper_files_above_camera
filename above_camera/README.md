# Helpers files for the task

- *camera_set_up.py*  simple file complete of map creation, real-time robot position and synchronisation processes
- *check_map.py* shows the last created map
- *check_orientation* streaming of the camera while returning positional information of the ArUco
- *circular_path.py* controller having the robot turn in a circular path 
- *vision.py* class allowing the read of ArUco information
